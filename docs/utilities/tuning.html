PID Tuning Tools

Auto-tuning and live parameter adjustment.

SimpleTune (Auto-Tuner)

Implements the Åström–Hägglund Relay method for automatic PID tuning. This "Bang-Bang" approach estimates the system's Ultimate Gain (Ku) and Ultimate Period (Tu) to calculate stable PID constants.

// Configure Tuner (Setpoint, Relay Voltage, Hysteresis)
SimpleTune tuner = new SimpleTune(100.0, 6.0, 2.0);
tuner.start();

// In Periodic Loop
if (!tuner.isFinished()) {
    double output = tuner.update(currentPosition);
    motor.setVoltage(output);
} else {
    // Tuning Complete
    PIDResult result = tuner.calculateConstants(TuningRule.ZIEGLER_NICHOLS);
    System.out.println("kP: " + result.kP + ", kI: " + result.kI + ", kD: " + result.kD);
}

Tuning Rules

Supports multiple commonly used tuning formulas:

  • ZIEGLER_NICHOLS: Aggressive, fast response.
  • TYREUS_LUYBEN: More conservative, good for laggy systems.
  • NO_OVERSHOOT: Designed to prevent overshooting the setpoint.
  • P_ONLY / PI_ONLY: Restricts to specific terms.

LoggedTunableNumber

A wrapper around double that syncs with AdvantageScope/SmartDashboard for live tuning. Changes in the dashboard instantly update the robot's code.

LoggedTunableNumber kP = new LoggedTunableNumber("Elevator/kP", 0.5);

public void periodic() {
    if (kP.hasChanged(hashCode())) {
        pidController.setP(kP.get());
    }
}
