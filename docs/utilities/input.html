Input Wrappers

Simplified access to Xbox controllers and joysticks.

XBoxWrapper

A strongly-typed wrapper for the standard FRC XboxController. Exposes buttons as JoystickButton objects for easy binding.

// Initialize
XBoxWrapper driver = new XBoxWrapper(0);

// Bind Commands
driver.A.onTrue(new ShootCommand());
driver.RB.whileTrue(new IntakeCommand());

// Read Axis
double speed = driver.getLeftY();
double turn = driver.getRightX();

// Rumble
driver.setRumble(RumbleType.kBothRumble, 1.0);
