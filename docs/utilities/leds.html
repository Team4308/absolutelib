LED Control

Efficient manipulation of Addressable LEDs (WS2812B).

Leds Class

Wraps WPILib's AddressableLED to provide high-level features like buffer views, patterns, and HSV support. Handles buffer management automatically.

Features

  • Buffer Views: Treat a section of the strip as its own independent LED strip.
  • Pattern Support: Apply dynamic patterns to the whole strip or specific views.
  • HSV Support: Set colors using Hue, Saturation, and Value for easier rainbow effects.

Example Usage

// 1. Initialize
Leds leds = new Leds(9, 60); // Port 9, 60 LEDs
leds.start();

// 2. Create Views (e.g., for different subsystems)
var intakeSection = leds.createBufferView(0, 20);
var shooterSection = leds.createBufferView(20, 20);
var climberSection = leds.createBufferView(40, 20);

// 3. Apply Colors/Patterns
leds.applyToView(intakeSection, Color.kGreen);
leds.applyHSVToView(shooterSection, 120, 255, 128); // Dim Green

// 4. Update
leds.update();
