Math Utilities

Essential mathematical helpers for robotics control logic.

DoubleUtils

Static utility class for common double operations. Null-safe and optimized for control loops.

Core Methods

  • clamp(val, min, max): Restricts a value within a range.
  • normalize(val): Clamps a value between -1.0 and 1.0 (useful for motor output).
  • mapRange(val, inMin, inMax, outMin, outMax): Linearly maps a value from one range to another.

Example

// Map joystick input (-1 to 1) to servo angle (0 to 180)
double servoAngle = DoubleUtils.mapRange(joystick.getY(), -1.0, 1.0, 0.0, 180.0);

// Ensure motor output never exceeds safety limits
double safeOutput = DoubleUtils.clamp(calculatedOutput, -0.8, 0.8);
