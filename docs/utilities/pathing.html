On-The-Fly Pathing

Dynamic path generation for PathPlanner.

Overview

Allows you to generate PathPlanner paths at runtime based on the robot's current pose and a dynamic target. Useful for auto-aligning to game pieces or avoiding dynamic obstacles.

Available Shapes

  • direct: Straight line (spline) to target.
  • arcViaOffsetMidpoint: Curved path via a left/right offset point.
  • sCurve: Double-bend path for complex alignments.
  • orbit: Circular path around a point of interest.
  • manhattan: Moves only along X then Y (or Y then X).

Example

// 1. Define Constraints
PathConstraints constraints = OnTheFlyPathing.constraints(3.0, 2.0);

// 2. Generate Path
PathPlannerPath path = OnTheFlyPathing.arcViaOffsetMidpoint(
    currentPose, 
    targetPose, 
    0.5, // 0.5m offset (Curve Left)
    constraints, 
    targetPose.getRotation(), // End Heading
    0.0, // End Velocity
    false
);

// 3. Execute
AutoBuilder.followPath(path).schedule();
