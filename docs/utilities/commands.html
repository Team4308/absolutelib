Command Utilities

Enhanced command-based programming features.

AbsoluteCommand

Wrapper for WPILib's Command class that adds absolute timeout capabilities. Useful for ensuring commands eventually finish even if their isFinished() condition hangs.

Usage

// Wrap an existing command with a 2-second timeout
Command autoShoot = AbsoluteCommand.withTimeout(new ShootCommand(), 2.0);

// Or subclass it
public class TimedIntake extends AbsoluteCommand {
    public TimedIntake() {
        super(new IntakeCommand(), 1.5); // 1.5s timeout
    }
}

It can also use a dynamic timeout supplier if the duration depends on sensor data.
