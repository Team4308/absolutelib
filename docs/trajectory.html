Trajectory Solver (2026 REBUILT)

The crown jewel of the 2026 update. A complete ballistic solver that accounts for air resistance, Magnus effect, and field obstacles. It supports multiple solve modes and integrates with the ShooterSystem.

Key Features

  • Physics-Based: Accounts for gravity, drag, and Magnus effect via RK4 simulation.
  • Obstacle Avoidance: Checks trajectory against defined field obstacles (walls, stage, etc.).
  • Multiple Modes: CONSTRAINT (fast, algebraic) and SWEEP (thorough, iterative) modes.
  • Flywheel Simulation: Integrated FlywheelConfig for accurate spin-up modeling and exit velocity calculation.
  • Builder APIs: Fluent builders for SolverConfig, ShotInput, and TrajectorySolver.

Usage

// 1. Configure the solver
TrajectorySolver.SolverConfig config = TrajectorySolver.SolverConfig.builder()
    .minPitchDegrees(25)
    .maxPitchDegrees(60)
    .maxRpm(6000)
    .build();

TrajectorySolver solver = TrajectorySolver.builder()
    .gamePiece(GamePieces.REBUILT_2026_BALL)
    .config(config)
    .build();

// 2. (Optional) Attach flywheel for simulation
FlywheelConfig flywheel = new FlywheelConfig(
    FlywheelConfig.FlywheelType.DUAL_OVER_UNDER,
    GamePieces.REBUILT_2026_BALL,
    4.0, FRCMotors.KRAKEN_X60, 1.0
);
solver.setFlywheel(flywheel);

// 3. Create Input using builder
ShotInput input = ShotInput.builder()
    .shooterPositionMeters(robotX, robotY, shooterHeight)
    .targetPositionMeters(targetX, targetY, targetZ)
    .robotVelocity(robotVx, robotVy)
    .shotPreference(ShotInput.ShotPreference.AUTO)
    .collisionCheckEnabled(true)
    .build();

// 4. Solve
TrajectoryResult result = solver.solve(input);

if (result.isSuccess()) {
    double pitch = result.getPitchAngleDegrees();
    double rpm = result.getRecommendedRpm();
    // Aim and Fire!
}

Solver Configuration Presets

// Default — balanced speed/accuracy
SolverConfig config = SolverConfig.defaults();

// High Accuracy — finer tolerances, detailed flywheel gen
SolverConfig config = SolverConfig.highAccuracy();

// Quick Solve — coarser tolerances, fast for real-time
SolverConfig config = SolverConfig.quickSolve();

Obstacle Avoidance

The solver can automatically reject trajectories that collide with field elements. Obstacles are defined in ObstacleConfig and passed to the ShotInput.

ShotInput input = ShotInput.builder()
    .shooterPositionMeters(robotX, robotY, shooterHeight)
    .targetPositionMeters(targetX, targetY, targetZ)
    .collisionCheckEnabled(true)
    .addObstacle(stageObstacle)
    .build();
